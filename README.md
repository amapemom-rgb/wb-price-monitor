# WB Price Monitor

Простой мониторинг цен конкурентов на Wildberries на базе **Google Sheets + Apps Script**.
Скрипт раз в 2 часа берёт цены из публичного API WB и, если цена изменилась,
пишет в лог и шлёт уведомление в Telegram.

## Архитектура

```
Лист «Товары» (артикулы)
        |
        v
   checkPrices()  --батч-запрос-->  card.wb.ru/cards/v2/detail
        |                                   |
        v                                   v
 сравнение с предыдущей ценой      текущая цена
        |
        +-- изменилась -> лист «Лог» + уведомление в Telegram
        +-- нет -> ничего

Триггер каждые 2 часа · ошибки прогона -> письмо от Google на почту
```

## Структура таблицы

**Лист «Товары»** — ты заполняешь только столбцы A и B, остальное скрипт заполняет сам:

| A Артикул | B Метка | C Название | D Пред.цена | E Тек.цена | F Изменение | G Обновлено |
|-----------|---------|------------|--------------|------------|-------------|-------------|
| 12345678  | моё     | (авто)     | (авто)       | (авто)     | (авто)      | (авто)      |
| 87654321  | конкурент| (авто)     | (авто)       | (авто)     | (авто)      | (авто)      |

**Лист «Лог»** — создаётся автоматически: Дата/время · Артикул · Название · Старая цена · Новая цена · Изменение.

## Быстрый старт

См. [SETUP.md](SETUP.md). Коротко: создать таблицу -> вставить `code.gs` -> завести
`TELEGRAM_TOKEN` и `TELEGRAM_CHAT_ID` в Script Properties -> запустить `createTrigger()` один раз.

## Важные оговорки

- **Неофициальный API.** `card.wb.ru` — недокументированный эндпоинт, WB может
  менять его структуру. `extractPriceU()` устойчив к основным вариантам поля цены.
- **Цена ≈ витрина, не персональная.** Персональная СПП и скидка WB-кошелька не
  учтены — API отдаёт базовую/витринную цену.
- **Категория 18+.** Карточки для взрослых могут отдаваться иначе — проверьте их отдельно.
- **ToS.** Парсинг неофициального API формально нарушает правила WB (п. 9.9.6).
- **Свои товары** надёжнее мониторить через официальный API цен WB.
