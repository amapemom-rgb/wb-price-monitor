# WB Price Monitor

Простой мониторинг цен конкурентов на Wildberries на базе **Google Sheets + Apps Script**.
Скрипт раз в 2 часа берёт цены у API WB и, если цена изменилась, пишет в лист «Лог»
и шлёт уведомление в Telegram.

## Архитектура

```
Лист «Товары» (артикулы)
        |
        v
   checkPrices()  --батч-запрос-->  API Wildberries (v4)
        |                                   |
        v                                   v
 сравнение с предыдущей ценой      текущая цена
        |
        +-- изменилась -> лист «Лог» + уведомление в Telegram
        +-- нет -> ничего (бот молчит)

Триггер каждые 2 часа · ошибки прогона -> письмо от Google на почту
```

## Структура таблицы

**Лист «Товары»** — 8 колонок. Ты заполняешь только **B (Артикул)**; A (Комментарии)
и C (Метка) — по желанию. Остальное скрипт заполняет сам:

| A Комментарии | B Артикул | C Метка | D Название | E Пред.цена | F Тек.цена | G Изменение | H Обновлено |
|---------------|-----------|---------|------------|-------------|------------|-------------|-------------|
| (по желанию)  | 405799190 | моё     | (авто)     | (авто)      | (авто)     | (авто)      | (авто)      |
| (по желанию)  | 314985537 | конкурент | (авто)   | (авто)      | (авто)     | (авто)      | (авто)      |

Заголовки можно создать автоматически: запусти один раз функцию `initSheet`.

**Лист «Лог»** — создаётся автоматически: Дата/время · Артикул · Название · Старая цена ·
Новая цена · Изменение. Важно: сюда добавляется строка **только когда цена реально
изменилась**. Это история изменений, а не журнал всех запусков — поэтому лист растёт
редко и это нормально. «Товары» показывает текущее состояние, «Лог» — хронологию.

## Быстрый старт

См. [SETUP.md](SETUP.md). Коротко: создать таблицу -> вставить `code.gs` -> запустить
`initSheet` (заголовки) -> завести `TELEGRAM_TOKEN` и `TELEGRAM_CHAT_ID` в Script
Properties -> вписать артикулы в столбец B -> запустить `createTrigger` один раз.

## Важные оговорки

- **Неофициальный API.** Скрипт использует внутренний эндпоинт WB
  (`www.wildberries.ru/__internal/u-card/cards/v4/detail`). Он недокументирован и может
  измениться без предупреждения — это самое хрупкое место проекта. При смене формата
  ответа скрипт падает с ошибкой (не пишет тихо пустоту), и Google присылает письмо.
- **Цена ≈ витрина, не персональная.** Персональная СПП и скидка WB-кошелька не учтены —
  API отдаёт базовую/витринную цену.
- **Категория 18+.** Карточки для взрослых могут отдаваться иначе — проверьте их отдельно.
- **ToS.** Парсинг неофициального API формально нарушает правила WB (п. 9.9.6).
- **Свои товары** надёжнее мониторить через официальный API цен WB.
