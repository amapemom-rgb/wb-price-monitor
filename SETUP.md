# Пошаговая настройка

## 1. Создай Telegram-бота

1. Открой в Telegram [@BotFather](https://t.me/BotFather) -> `/newbot` -> придумай имя.
2. BotFather вернёт **токен** вида `123456:AAE...xyz` — сохрани его.
3. Напиши своему боту любое сообщение (чтобы чат существовал).
4. Узнай **chat_id**: открой в браузере
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates` -> найди `"chat":{"id":...}`.
   (Альтернатива: бот [@userinfobot](https://t.me/userinfobot) покажет твой id.)

## 2. Создай таблицу

1. Новая Google Таблица -> переименуй первый лист в **Товары**.
2. В первой строке заголовки:
   `Артикул | Метка | Название | Пред.цена | Тек.цена | Изменение | Обновлено`
3. С второй строки впиши артикулы (столбец A) и метку «моё/конкурент» (B).
   Остальные столбцы скрипт заполнит сам.

> Артикул (nm) — число из ссылки на товар:
> `wildberries.ru/catalog/`**`12345678`**`/detail.aspx`.

## 3. Вставь код

1. В таблице: **Расширения -> Apps Script**.
2. Удали содержимое `Code.gs` и вставь целиком файл [`code.gs`](code.gs).
3. Сохрани.

## 4. Заведи секреты (Script Properties)

1. В редакторе Apps Script: **Project Settings** (шестерёнка слева).
2. Раздел **Script Properties -> Add script property**. Добавь два:
   - `TELEGRAM_TOKEN` = твой токен бота
   - `TELEGRAM_CHAT_ID` = твой chat_id
3. Сохрани.

> НИКОГДА не вписывай токен прямо в код — репозиторий публичный.

## 5. Проверь связку

1. В редакторе выбери функцию `testTelegram` -> **Run**.
2. При первом запуске Google попросит разрешить доступ — разреши.
3. В Telegram должно прийти «✅ связь работает».
4. Затем запусти `checkPrices` вручную — таблица заполнится (первый прогон —
   только фиксирует базовые цены, без алертов).

## 6. Включи расписание

1. Выбери функцию `createTrigger` -> **Run** ОДИН раз.
   Это создаст триггер «каждые 2 часа».
2. Проверь: **Triggers** (будильник слева) — должен быть триггер на `checkPrices`.

## 7. Сигналка об ошибках (без кода)

В **Triggers -> … (три точки у триггера) -> настройки уведомлений** выбери
«уведомлять немедленно». Тогда при любом падении скрипта Google сам пришлёт
письмо на твою почту.

## Как добавлять артикулы позже

Просто допиши новую строку в лист «Товары» (столбец A — артикул, B — метка).
Код менять не нужно.
