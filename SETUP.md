# Пошаговая настройка

## 1. Создай Telegram-бота

1. Открой в Telegram [@BotFather](https://t.me/BotFather) -> `/newbot` -> придумай имя.
2. BotFather вернёт **токен** вида `123456:AAE...xyz` — сохрани его.
3. Напиши своему боту любое сообщение (чтобы чат существовал).
4. Узнай **chat_id**: открой в браузере
   `https://api.telegram.org/bot<ТОКЕН>/getUpdates` -> найди `"chat":{"id":...}`.
   (Альтернатива: бот [@userinfobot](https://t.me/userinfobot) покажет твой id.)

## 2. Создай таблицу и вставь код

1. Новая Google Таблица -> Расширения -> Apps Script.
2. Удали содержимое `Code.gs` и вставь целиком файл [`code.gs`](code.gs). Сохрани.
3. Запусти функцию `initSheet` один раз — она создаст лист **Товары** с заголовками
   (8 колонок) и лист **Лог**.

**Структура листа «Товары»:**
`A Комментарии | B Артикул | C Метка | D Название | E Пред.цена | F Тек.цена | G Изменение | H Обновлено`

Ты заполняешь только столбец **B (Артикул)**; A и C — по желанию. Остальное скрипт
заполняет сам.

> Артикул (nm) — число из ссылки на товар:
> `wildberries.ru/catalog/`**`405799190`**`/detail.aspx`.

## 3. Заведи секреты (Script Properties)

1. В редакторе Apps Script: **Project Settings** (шестерёнка слева).
2. Раздел **Script Properties -> Add script property**. Добавь два:
   - `TELEGRAM_TOKEN` = твой токен бота
   - `TELEGRAM_CHAT_ID` = твой chat_id
3. Сохрани.

> НИКОГДА не вписывай токен прямо в код — репозиторий публичный.

## 4. Проверь связку

1. В редакторе выбери функцию `testTelegram` -> **Run**.
2. При первом запуске Google попросит разрешить доступ — разреши.
3. В Telegram должно прийти «✅ связь работает».
4. Впиши артикулы в столбец B листа «Товары».
5. Запусти `checkPrices` вручную — таблица заполнится (первый прогон по каждому
   товару только фиксирует базовую цену, без алертов).

## 5. Включи расписание

1. Выбери функцию `createTrigger` -> **Run** ОДИН раз. Создаст триггер «каждые 2 часа».
2. Проверь: **Триггеры** (будильник слева) — должен быть триггер на `checkPrices`.

## 6. Сигналка об ошибках (без кода)

В **Триггеры -> … (три точки у триггера) -> настройки уведомлений** выбери
«уведомлять немедленно». Тогда при любом падении скрипта Google сам пришлёт письмо
на твою почту.

## Как проверить, что всё работает

- Лист «Товары»: колонка «Обновлено» должна **сдвигаться каждые 2 часа**, даже если
  цены не менялись. Это доказывает, что расписание живо.
- Тест алертов «на честность»: вручную впиши в ячейку «Тек.цена» (столбец F) любого
  товара другое число и запусти `checkPrices`. Должно прийти ровно одно уведомление
  и добавиться одна строка в «Лог». На следующем прогоне скрипт вернёт реальную цену.

## Как добавлять артикулы позже

Просто допиши новую строку в лист «Товары» (столбец B — артикул). Код менять не нужно.
