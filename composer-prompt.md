# Промпт для Composer

Скопируй текст ниже целиком и отдай Composer. Код берётся из этого
репозитория (`code.gs`) — его Composer не переписывает, а только вставляет
и настраивает.

---

**Роль:** ты настраиваешь готовый проект Google Sheets + Apps Script в браузере.
Код уже написан и лежит в репозитории
`github.com/amapemom-rgb/wb-price-monitor` (файл `code.gs`). Твоя задача —
**не переписывать код**, а развернуть его и проверить.

**Задача:** мониторинг цен конкурентов на Wildberries. Скрипт читает артикулы из
листа «Товары», берёт цены у публичного API WB, при изменении пишет в лист
«Лог» и шлёт уведомление в Telegram. Запуск каждые 2 часа.

**Что нужно сделать в браузере:**

1. Создать Google Таблицу, первый лист назвать **Товары** с заголовками:
   `Артикул | Метка | Название | Пред.цена | Тек.цена | Изменение | Обновлено`.
2. Открыть Расширения -> Apps Script, вставить содержимое `code.gs` из репо.
3. В Project Settings -> Script Properties завести `TELEGRAM_TOKEN` и
   `TELEGRAM_CHAT_ID` (значения даст пользователь). НЕ вписывать их в код.
4. Запустить `testTelegram`, убедиться что сообщение пришло.
5. Запустить `checkPrices` вручную (первый прогон — базовые цены).
6. Запустить `createTrigger` один раз — создаст триггер «каждые 2 часа».
7. В настройках триггера включить уведомления об ошибках «немедленно» (на почту).

**Ревью безопасности (важно):**
Пройдись по проекту и **найди слабые места, объясни риск и как его закрыть**.
Обязательно проверь:
- Нет ли секретов (токен, chat_id) в коде или в истории репозитория. Они
  должны быть только в Script Properties.
- Кто имеет доступ к таблице и к проекту Apps Script (не открыт ли доступ «всем
  по ссылке»).
- Обработка ошибок: что будет, если API WB недоступен, вернёт пустоту или
  сменит формат ответа.
- Нет ли риска бана по частоте/объёму запросов.
- Корректность прав доступа (OAuth scopes), которые запрашивает скрипт.

По каждому пункту — конкретная рекомендация, как устранить. Не вноси
изменения в логику без моего подтверждения — сначала покажи список находок.
