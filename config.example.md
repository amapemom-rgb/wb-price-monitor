# Шаблон конфигурации

Скрипт НЕ хранит секреты в коде. Все чувствительные значения задаются в
**Script Properties** (редактор Apps Script → шестерёнка «Project Settings» →
раздел «Script Properties»).

Нужно завести два свойства:

| Имя свойства      | Значение                                             |
|--------------------|-----------------------------------------------------|
| `TELEGRAM_TOKEN`   | токен бота от @BotFather, напр. `123456:AAE...xyz`     |
| `TELEGRAM_CHAT_ID` | id чата/канала, куда слать, напр. `987654321`      |
| `WB_API_KEY`       | ВБ_API_Ключ категории «Цены и скидки»              |

Необязательно: геозона задаётся константой `WB_DEST` в начале `code.gs`.

> Артикулы конкурентов хранятся в самой таблице (лист «Товары»),
> а не в коде — поэтому публичный репозиторий не раскрывает стратегию.
